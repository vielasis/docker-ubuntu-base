## Ubuntu Docker Image

A barebones Ubuntu Docker image containing minimal packages for development

##### Included packages:
- `curl`
- `wget`
- `rsync`
- `sudo`
- `expect`
- `git`
- `python` (2.7.x)
- `build-essential`
- `tree`
